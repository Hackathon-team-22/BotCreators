# Domain Glossary

| Термин | Определение |
|--------|-------------|
| Telegram-чат | Группа или супергруппа в Telegram, в которой ведётся обсуждение и публикуются сообщения. |
| Экспорт чата | История чата, выгруженная пользователем в мобильном клиенте Telegram (JSON/HTML/ZIP). |
| Сессия экспорта | Набор файлов экспорта, отправленных одним пользователем в рамках обработки ботом. |
| Сообщение чата | Нормализованная запись с автором, текстом, упоминаниями и метаданными. |
| Участник | Пользователь, который писал сообщения в данном чате (идентифицируется по user_id/username/display name). |
| Упомянутый пользователь | Пользователь, который упоминался в сообщениях, даже если сам не писал. |
| Канал | Telegram-канал, упомянутый или выступающий источником форвардов. |
| AudienceProfile | Абстракция субъекта аудитории (участник, упомянутый, канал) с единым идентификатором `ProfileId`. |
| ExtractionResult | Итоговый набор нормализованных профилей, разделённых на участников, упомянутых и каналы. |
| AudienceReport | Итоговый отчёт: текстовый список или Excel-файл, сформированный на основе ExtractionResult. |
| ReportPolicy | Правило выбора формата отчёта (текст или Excel) в зависимости от размера аудитории и конфигурации. |
| Conversation Service | Прикладной сервис, управляющий пользовательскими сессиями, загрузкой файлов и командами Telegram. |
| Pipeline | Последовательность Parse → Extract → Report, проверяющая лимиты по количеству файлов, размеру и времени обработки. ***
